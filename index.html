PROJECTS

Camera Slider

Created a mobile-controlled camera slider that moves based on the distance and time interval set by the user on the app. All backend requests are handled by a Raspberry Pi, and a servo-motor controls the movement of the slider. The app was built using SwiftUI, and the connection between the app and the Pi was established using BLE technology.

NetGuard

Built an IoT security system that detects DDoS attacks and protects IoT devices by shutting down all ports, notifying the user of the attack and then uploading the attack information to an Azure database. Users can then reconfigure the device through a custom SSH key generated by this system. Developed using Python, Docker, Azure, Twilio and Remote.it.

Workout Generator

Developed an iOS fitness app that generates a custom workout for the user based on the muscle groups that they select. These workouts, along with physical stats like weight and height, are saved on Firebase. Firebase Authentication is also used to identify each user.
